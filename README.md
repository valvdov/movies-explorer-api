# API для дипломного проекта для Яндекс.Практикум

### Испиользуемые технологии:
+ NodeJS
+ Express
+ MongoDB
+ Mongoose

### Сделанный функционал:

+ Регистрация
+ Авторизация
+ Обновление данных пользователя
+ Получение информации о текущем пользователе
+ Получение списка фильмов
+ Создание фильма
+ Удаление фильма
